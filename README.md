# AI Assisted Job Application Tracker

Kanban based job tracker with AI powered JD parsing and resume suggestions.

## Tech Stack

MERN + TypeScript + Tailwind + OpenAI API

## Setup

clone repo

cp .env.example .env

fill env values

### install dependencies
npm run install-all

### start app
npm run dev

server runs on:
http://localhost:5000

client runs on:
http://localhost:5173

## environment variables

MONGO_URI=
JWT_SECRET=
OPENAI_API_KEY=

## features

authentication
kanban board
AI job description parsing
resume bullet suggestions
drag drop status updates
